# Imagens da seção Currently

Cada arquivo deve ter o mesmo nome passado em imageSrc no currentlyData.items.
Exemplo: ddia.jpg → imageSrc: '/static/images/favorites/ddia.jpg'

Tamanho ideal: 280×140 px (proporção 2:1, landscape)
Formato: JPG ou WebP
Peso máximo recomendado: 80 KB por imagem

A imagem é exibida com object-cover, altura fixa de 140 px, largura 100%.
Imagens verticais serão cortadas — prefira capas horizontais ou quadradas.
